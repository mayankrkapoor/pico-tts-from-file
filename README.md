SVOX Pico TTS - Read text from file
=====================================

### Description

This script read the text from a file and convert it into a sound file.


### Getting started
```
  $ ./picoTTSFromFile.sh text_en.txt
  $ ./picoTTSFromFile.sh de-DE text_de.txt  
```
